# 🚀 Hi there, I'm Hatano (@Hatano123)

豊橋技術科学大学の3年生（情報工学専攻）です。豊田高専を卒業後、現在は **Neural Radiance Fields (NeRF)** をはじめとする画像処理や深層学習領域の研究に取り組んでいます（高専時代はセンサーフュージョンによる高精度物体検出を研究）。
